# 🎨 Eleanor Lian - Complete Interactive Experience

**A stunning, professional website showcasing all Eleanor Lian characters, stories, and magical worlds.**

## ✨ Features

### 🖼️ **Complete Image Gallery**
- All character images from the Eleanor Lian universe
- Lightbox viewer with smooth animations
- Organized by story and character

### 👥 **Interactive Character Browser**
- Detailed character cards with biographies
- Powers, relationships, and story arcs
- Filter and search functionality

### 📖 **Story Explorer**
- Complete information about all 3 stories
- Statistics and key moments
- Character appearances

### 🗺️ **Magical Worlds**
- Descriptions of each magical realm
- Magic systems and unique features
- Interactive maps (coming soon)

### 💾 **Offline Ready**
- Save the page and access without internet
- Perfect for presentations and demos
- All assets included locally

## 🚀 Quick Deploy

### **Option 1: Vercel (Recommended - 30 seconds)**
1. Click the "Deploy with Vercel" button above
2. Sign in with GitHub
3. Click "Deploy"
4. Your site is live at `your-site.vercel.app`

### **Option 2: Netlify (Drag & Drop)**
1. Click the "Netlify Deploy" button above
2. Connect your GitHub account
3. Your site is live immediately

### **Option 3: Manual Hosting**
1. Download this repository
2. Upload to any web server
3. Open `index.html` in browser
