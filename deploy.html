📦 How to Deploy (Step by Step)

Option 1: Vercel (Recommended - 30 seconds)

  1. Go to vercel.com
  2. Sign up with GitHub, GitLab, or email (free)
  3. Click "New Project"
  4. Drag and drop the eleanor-complete folder
  5. Click "Deploy"
  6. Your site is live at your-site.vercel.app
# Or use Vercel CLI
npm i -g vercel
cd eleanor-complete
vercel
vercel --prod

Option 2: Netlify (Drag & Drop)

  1. Go to netlify.com/drop
  2. Drag the eleanor-complete folder to the page
  3. That's it! Site is live immediately
  4. Customize domain in site settings
# Or use Netlify CLI
npm install -g netlify-cli
netlify deploy --prod --dir=.

Option 3: Manual Hosting

  1. Download the ZIP file
  2. Extract to any web server
  3. Upload to: GitHub Pages, GitLab Pages, Firebase, AWS, etc.
  4. Or open index.html directly in browser
